# PresenceCraft
- This Fabric mod add Discord RPC for Minecraft
## What is news?
- 1.0.1:
    - Change the LargeImage according to your dimensions.
    - Removed Dimensions details text.
    - Added LargeImageText will shown your Minecraft version.
    - Added SmallImageText will shown your username.
    - Added show player in your multiplayer world.
    - Added Config feature (you can customize which one is displayed)
    - Added Mod icon
## Instalation
- You can install at [release page](https://github.com/hunggisagoner/PresenceCraft/releases) or [auto build](https://github.com/hunggisagoner/PresenceCraft/actions)
